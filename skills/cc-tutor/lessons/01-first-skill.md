# L1. 첫 번째 Skill 만들기

## 이 레슨의 목표
- Skill이 뭔지 이해하기
- 그걸 어떻게 만드는지 체험하기

---

## 개념: Skill = 규칙을 파일에 적은 것

```
~/.claude/skills/mail-polish/
└── SKILL.md
```

이 파일 안에:
- **이름** (`name`)
- **언제 쓰는지** (`description`)  
- **규칙들** (본문)

그럼 끝. `/mail-polish` 명령어로 발동돼요.

---

## 튜터의 질문 1️⃣

**SKILL.md에 반드시 들어가야 할 건 뭘까요?**

선택지:
- 이름과 설명
- 프로그래밍 코드
- 버전 번호

(생각해보고 답해 봐요)

---

## 튜터의 질문 2️⃣

**규칙을 글로 써 둔다... 근데 꼭 지켜질까요?**

예: "메일은 반드시 '안녕하세요'로 시작해" ← 이게 정말 지켜질까?
