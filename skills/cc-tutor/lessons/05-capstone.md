# L5. 졸업과제 — 내 스킬 만들기

## 이 레슨의 목표
- L1~L4 배운 것 전부를 써서 **내 업무 스킬 1개** 만들기
- 직접 만들어 보면서 이해하기

---

## 과제: 당신 업무에서 스킬 1개 고르기

**조건:**
- 반복하는 일이어야 함
- 규칙이 정해져 있어야 함
- 결과물이 파일이어야 함 (메일, 문서, 보고서 등)

---

## 튜터의 질문 1️⃣

**당신 업무에서 "매주 또는 매일 반복하는 일"은 뭔가요?**

(아무거나, 대충 말해 봐요)

---

## 튜터의 질문 2️⃣

**그 일의 규칙을 3가지만 꼽아볼 수 있나요?**

예: "메일은 반드시 이렇게 시작 / 금액은 천단위 쉼표 / 마지막은 이렇게"

---

## 만드는 순서

1. Claude에게 한마디: **"○○ 스킬 만들어줘, ~/.claude/skills에 저장해줘"** (폴더·파일은 Claude가 알아서)
2. SKILL.md에 규칙을 자연어로 적기 (정답 구조 없음 — 내 의도대로)
3. 발동해서 테스트
4. 검증이 오류를 잡는지 확인

---

## 완성했을 때

당신의 스킬이 **발동되면 규칙대로 알아서 처리해주는** 경험을 할 거예요.
