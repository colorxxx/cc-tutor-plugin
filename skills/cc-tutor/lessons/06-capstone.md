# L6. 졸업과제 — 나의 업무 스킬, 처음부터 직접 만들기

## 학습 목표
- 과정 전체에서 **유일하게 처음부터 직접** 자기 업무 스킬 1개를 완성한다.
- "좋은 스킬 3요소"(천장/바닥/문) 기준으로 자기 스킬을 점검·개선한다.

## 진행 방식 (튜터 노트)
이 레슨은 강의가 아니라 **프로젝트 코칭**이다. 여기서부터는 튜터가 대신 만들어주지 않는다 — 묻고, 힌트만 준다 (힌트 → 더 구체적 힌트 → 정답 일부 순서로).
든든한 출발점: 학습자는 이미 preset 4개를 굴리고 고쳐봤다. **"notice-maker를 뜯어 고치듯 만들면 된다"** 고 안심시킬 것.

### 1단계: 주제 선정
- L5에서 기록해둔 "반복 작업" 메모부터 확인. 없으면 인터뷰:
  - "지난주에 두 번 이상 반복한 문서/조사/정리 작업은?"
  - "신입에게 시키면 반나절 걸리지만 규칙은 정해져 있는 일은?"
- 좋은 과제 조건: ① 반복적 ② 규칙 설명 가능 ③ 결과물이 파일/문서 ④ 작게 (첫 버전은 30분 내 시연 규모).

### 2단계: 설계 (만들기 전에 계획 — plan 모드 정신)
학습자가 답하게 한다 (preset과 짝지어 물으면 쉬움):
1. 입력/출력: 무엇을 받아 무엇이 나오나?
2. 암묵지: references/에 뭘 넣나? (← memo-clean의 용어집처럼)
3. 양식: templates/가 필요한가? (← notice-maker의 안내문양식처럼)
4. 도구: 외부 도구(MCP나 WebFetch)가 필요한가? (없어도 됨 — docs-lookup 참고)
5. 바닥: 뭐가 틀리면 사고인가? → 검증 체크리스트 3개

### 3단계: 구축
- 시작은 골격 복사부터 권장: `cp -r ${CLAUDE_PLUGIN_ROOT}/skills/cc-tutor/templates/my-first-skill ~/.claude/skills/<스킬명>`
- 권장 순서: SKILL.md 뼈대 → references → templates → 검증 체크리스트.
- 학습자가 직접 작성. 단, "내가 말로 설명할 테니 md로 정리해줘"라고 Claude를 부려서 만드는 건 OK (그것도 활용!).

### 4단계: 시연 + 졸업 리뷰
- 실제 입력으로 발동 → 결과 확인 → **일부러 정보 빠진 입력**으로 한계 테스트 (notice-maker에서 본 그 장면을 자기 스킬로 재현).
- 튜터 리뷰 (templates/checklist.md L6 기준):
  - **천장**: 지시가 구체적인가? / **바닥**: 검증이 실제로 오류를 잡는가? / **문**: description만 보고 언제 쓰는지 알겠는가?
  - 개선점은 최대 3개만 → 1개는 그 자리에서 같이 고치기.
- 통과 시: dashboard에 수료 기록 + 축하 + 성장 경로 안내:
  - 스킬이 늘면 command 단축키 / "항상 켜두는" 워크플로가 되면 agent 승격 (`claude --agent`)
  - 공식 문서: code.claude.com/docs (skills / sub-agents / mcp)

## 퀴즈 (졸업시험 — 종합 5문제, 4개 이상 합격)

**Q1. 나만 아는 업무 지식을 스킬에 넣는 표준 방법은?**
- ① 본문에 전부 쓰기 / ② references/ 파일로 번들 ✅ / ③ 매번 대화로 설명 / ④ description에 넣기

**Q2. 결과물 양식을 고정하는 방법은?**
- ① templates/ 양식 파일 + "이대로 따르라" ✅ / ② 매번 예시를 말로 / ③ AI 재량 / ④ 폰트 설정

**Q3. "좋은 스킬 3요소"가 아닌 것은?**
- ① 정확한 프롬프트(천장) / ② 결과 검증(바닥) / ③ 발동(description) / ④ 화려한 디자인 ✅

**Q4. AI가 외부 시스템(메일, 법령DB)을 만지게 하는 연결 표준은?**
- ① HTML / ② MCP ✅ / ③ PDF / ④ VPN

**Q5. "격리된 작업대에서 일하고 마지막 보고만 돌려주는 보조 일꾼"은?**
- ① command / ② frontmatter / ③ subagent ✅ / ④ template

미달이면 틀린 개념의 레슨 복습 권한 뒤 재시험.
